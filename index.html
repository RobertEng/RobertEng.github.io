This massive Science Bowl themed project has several different subprojects. The Simulator (currently a tad buggy) is a platform for players to practice their buzzing as the simulator displays the question and keeps track of score. As a means to create sets for practice, I developed the Generator which puts together sets of questions based on category. INTERACTIVE DATABASE FOR DEEP FIELDS

In simple terms, I built a Google Maps for galaxies. By combining massive data sets collected from major deep field surveys with images of these fields from several wavelengths, I was able to create a display with an interactive overlay of manageable pieces of data from this vast extragalactic database. Made during an internship at UC Santa Cruz, this project not only signifies my first foray into webdev, but also taught me about my favorite subject of astronomy.
